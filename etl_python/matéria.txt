Aula 1 - Fundamentos de ETL
ETL - Extract (Extração), Transform (Transformação) e Load (Carregamento)
-Extract: os dados são extraídos de diferentes fontes de dados.
-Transform: os dados são propagados para a área de preparação de dados, onde são transformados e limpos.
-Load: os dados são carregados no data warehouse.

Formas de extração: Batch (extraídos em lote) e Streaming/Real Time (extraídos em tempo real)

Data Sources são as diferentes fontes de dados (sistema operacional, ERP, CRM, SQL...) -> Data Flow/Pipeline (processos: data validation, data cleaning, data transforming, data aggregating, data loading; enviados para um Data Warehouse) -> BI results (OLAP Analysis, Data Mining, Data Visualization, Reports, Dashborads.

Ferramentas/Pacotes: Apache Airflow, luigi, bonobo, bubbles, petl, pandas
